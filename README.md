# Multi-Model AI Chat Interface

Built an advanced AI chat interface capable of streaming responses from multiple models including Google Gemini, Kimi K2.5, and OpenAI.

## Features
- Streaming responses from multiple models
- Debugged routing issues (404s and empty error bubbles) between frontend and Express backend
- Resolved SSR pipeline errors
- Premium UI styling by purging blue tints for a strict, absolute black and neutral zinc aesthetic

## Tech Stack
- React
- Vite
- Express (cors)
- Vercel AI SDK (@ai-sdk/google, @ai-sdk/openai, ai)
- Playwright for testing
